Welcome to My Mid-2000s Website

This site has been upgraded from its original 90s look into a more modern mid-2000s design. Back then, websites started using cleaner layouts, external CSS, glossy buttons, rounded corners, and banner graphics.

This redesign focuses on replacing old HTML practices and table based layouts with proper semantic structure and CSS styling.

What's New in This 2000s Redesign

  • Modern layout using flexbox instead of tables
  • A banner image across the top of every page
  • Navigation styled as mid-2000s glossy buttons
  • A left sidebar with rounded corner blocks
  • External CSS file for all styling
