About This Mid-2000s Redesign

This page is part of my upgraded mid-2000s themed website project. The original version was built in a 1990s retro style using basic HTML, bright colours, and table-based layouts.

For this redesign, the goal was to modernise the structure using proper semantic HTML and external CSS, while still keeping the nostalgic feel of early to mid 2000s web design — glossy buttons, banner graphics, rounded corners, and cleaner layouts.

What This Page Used to Look Like

The original About page relied heavily on:

  • Inline <font> tags
  • Table based layout for positioning
  • Bright background colours
  • Old-school HTML attributes like bgcolor and vlink
